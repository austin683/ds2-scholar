**Illusory Talisman** is a type of [Item](/Items "Dark Souls 2 Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)**. **Illusory Talisman** , an item mentioned in the game code but not available in-game. [Unused Items](/Unused+Items "Dark Souls 2 Unused Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)** are Items listed in the game code, but not available in-game.

> A tiny charm embedded with imitative magic. Use to transform into a phantom. Use this talisman to trick invaders. Only provides the most superficial camouflage, but for those naïve enough to never question what they see, you should be very convincing."

### Illusory Talisman Usage in Dark Souls 2

  * Disguises the user as a phantom.

### Dark Souls 2 Illusory Talisman Location

  * Unavailable 

### Illusory Talisman Notes and Tips in Dark Souls 2

  * Removed from the final version of the game.
  * Replaced along with it's counterparts by the [Silver Talisman](/Silver+Talisman "Dark Souls 2 Silver Talisman")
  * Has the same effect as the [White Ring](/White+Ring "Dark Souls 2 White Ring")
