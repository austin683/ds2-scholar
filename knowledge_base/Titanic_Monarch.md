This is a player-created **Build** for Dark Souls 2.

  * Build Name: Titanic Monarch
  * Starting Class: Warrior/Knight
  * Starting Gift: Any
  * Build Focus: PvE

###  Build Equipment

  * **Right Hand 1: Greatsword**
  * **Right Hand 2: Bone Fist**
  * **Right Hand 3: Hand**
  * **Left Hand 1: Greatsword**
  * **Left Hand 2: Bone Fist**
  * **Left Hand 3: Hand**
  * **Head** : Drangleic Helm 
  * **Chest** : Drangleic Mail
  * **Hands** : Drangleic Gauntlets
  * **Legs** : Drangleic Leggings
  * **Spells: None**
  * Rings**: Royal Soldiers Ring +2, Chloranthy Ring +2, Ring of Giants +2, Ring of Blades +2 (Preferable but not required since this build is for Cosplay)**

  
  

##  Build Strategy

With Greatswords Powerstanced, there aren't very many PvE scenarios you'll find trouble with. Though you must take caution with cramped spaces and of the environment around you for if you hit an immovable object, you'll be left wide open for a punishing counter attack.
