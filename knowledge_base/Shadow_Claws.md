**Shadow Claws** are a type of [Item](/Items "Dark Souls 2 Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)**. **Shadow Claws** , a set of claws employed by those who thrive in the shadows. Each finger is equipped with a sharp blade. Those who forsake the knight's code engage in unspeakable deeds, driven by the allure of coin. [Unused Items](/Unused+Items "Dark Souls 2 Unused Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)** are Items listed in the game code, but not available in-game.

> Claws used by those who lurk in the shadows. A sharp blade is attached to each of the fingers. Those who abandoned the knight's code stopped to all manner of unspeakable deeds, all for a bit of coin"

### Shadow Claws Usage in Dark Souls 2

  * Cannot be picked up or equipped.

### Dark Souls 2 Shadow Claws Location

  * Chance to be dropped from one of the minions in the Flexile sentry boss fight (NG+)
  * When trying to pick up this item's hex code using debug program it will not allow player to pick up with the message of saying "inventory is full"
