This is a player-created **Build** for Dark Souls 2.

  * Build Name: [The Hero Faraam](https://fextralife.com/calculator/dks2builds.html?SiJ9qHRxD.DStQjUUkx.eIDvRc6V.KgCeMEDQ.EZxfilh4x.E7 "The Hero Faraam")
  * Starting Class: Warrior
  * Starting Gift: Human Effigy (though you can choose any)
  * Build Focus: PvE

###  Build Equipment

  * **Right Hand 1: Drakekeeper's Sword**
  * **Right Hand 2: Fume Sword**
  * **Right Hand 3: Black Dragon Shield (rampart golem shield is a good alternative)**
  * **Left Hand 1: Drakekeeper Sword**
  * **Left Hand 2: Sun Sword**
  * **Left Hand 3: Black Dragon Greatsword**
  * **Head** : Faraam Helm
  * **Chest** : Faraam Armor
  * **Hands** : Faraam Gauntlets
  * **Legs** : Faraam Boots
  * **Spells: None**
  * Rings**: Life Ring +3, Ring of Blades +2, Royal Soldiers Ring +2, Third Dragon Ring (Preferable but not required since this build is for Cosplay)**

##  Build Strategy

I find this build to be very effective in both PvP and PvE, while this class may look very weak, if you know how to roll properly then you should do just fine.
