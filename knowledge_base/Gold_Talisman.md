**Gold Talisman** is a type of [Item](/Items "Dark Souls 2 Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)**. **Gold Talisman** , an item mentioned in the game code but not found in-game. [Unused Items](/Unused+Items "Dark Souls 2 Unused Items") in [Dark Souls 2](/Dark+Souls+2+Wiki "Dark Souls 2 Wiki")**(DKS2)** are Items listed in the game code, but not available in-game.

> A tiny charm embedded with imitative magic. Use to transform into a treasure chest. Use this talisman to trick invaders. Only provides the most superficial camouflage, but for those desperate for quick riches, you will appear quite attractive indeed."

### Dark Souls 2 Gold Talisman Usage

  * Disguises the user as a treasure chest

### Gold Talisman Location in DKS2

  * Unavailable

### Gold Talisman Notes and Tips in Dark Souls 2

  * Unobtainable in the final version of the game
  * This along with it's counterparts were replaced by the [Silver Talisman](/Silver+Talisman "Dark Souls 2 Silver Talisman")
