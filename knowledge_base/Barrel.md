_"A large barrel. One can wear this barrel, but it conceals the arms, making movement very difficult._   
_The barrel is oddly comforting, however, and gradually restores HP, perhaps an effect of the strange crest painted upon it."_

###  Location

  * This cannot be acquired in game without using out of game tools.

###  Notes

  * The details of this item were lifted from the game files, and is just remnants of a beta item removed from the retail version.

###  Piece Information & Upgrades

First row = Regular. Second row = +10
